AZURE DATA FACTORY:

Azure Data Factory is Azure's cloud ETL service for scale-out serverless data integration and data transformation.
It offers a code-free UI for intuitive authoring and single-pane-of-glass monitoring and management.
You can also lift and shift existing SSIS packages to Azure and run them with full compatibility in ADF.

ACTIVITIES IN ADF

sno activity name description
1 copy you can copy data between two data stores
2 get metadata Get Metadata activity can be used to retrieve metadata of any data in Azure Data Factory.
3 look up Lookup activity can retrieve a dataset from any of the Azure Data Factory supported data sources.
4 web Web activity can be used to call a custom REST endpoint from a Data Factory pipeline.
5 append variable Append Variable activity could be used to add a value to an existing array variable defined in a Data Factory pipeline.
6 set variable Set Variable activity can be used to set the value of an existing variable of type String, Bool, or Array defined in a Data Factory pipeline.
7 filter Filter activity can be used in a pipeline to apply a filter expression to an input array.
8 for each The ForEach activity defines a repeating control flow in your pipeline.
9 if condition If Condition activity allows directing pipeline execution, based on evaluation of certain expressions.
